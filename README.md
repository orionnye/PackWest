# PackWest

TypeScript WebApp built on [@adobe/data](https://github.com/adobe/data) — Adobe Data Oriented Programming Library.

## Setup

```bash
npm install
```

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Build for production
- `npm run preview` — Preview production build

## @adobe/data

Uses data utilities (normalize, equals), observables (Observe.fromConstant, Observe.withMap), and ECS (Store, Vec3) from @adobe/data.
